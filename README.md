## 2. Deskripsi Singkat Project
Project ini merupakan eksperimen berbasis Python untuk mengimplementasikan operasi matematika dasar (penjumlahan, penggeseran, dan amplifikasi) pada sinyal diskrit 1D dan citra digital 2D. Eksperimen ini bertujuan untuk memahami dan membuktikan konsep superposisi serta sistem linier yang menjadi fondasi dalam Pengolahan Sinyal Digital (PSD).

## 3. Library yang Digunakan
- NumPy (untuk komputasi array dan operasi matematis)
- Matplotlib (untuk visualisasi plot grafik)
- OpenCV / cv2 (untuk membaca dan memproses citra)
- SciPy (untuk keperluan saintifik tambahan)

## 4. Cara Menjalankan Notebook
1. Clone atau unduh repository ini ke komputer lokal Anda.
2. Buka terminal/Command Prompt di dalam folder project, lalu instal library yang dibutuhkan dengan menjalankan perintah:
   `pip install -r requirements.txt`
3. Buka file Jupyter Notebook yang berada di folder `notebook/` menggunakan JupyterLab, VS Code, atau upload ke Google Colab.
4. Jalankan (*Run*) setiap *cell* kode secara berurutan dari atas ke bawah.
